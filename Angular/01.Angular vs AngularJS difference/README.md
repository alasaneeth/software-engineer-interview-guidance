## 1. Angular vs AngularJS Difference

| Feature | AngularJS (v1.x) | Angular (v2+) |
|---|---|---|
| **Language** | Plain JavaScript | TypeScript |
| **Architecture** | Model-View-Controller (MVC) | Component-Based |
| **Mobile Support** | No native mobile browser support | First-class mobile support |
| **Data Binding** | Two-way data binding | One-way and property/event binding |
| **Tooling** | Requires third-party tools | Built-in Angular CLI |
| **Status** | End-of-Life (No longer supported) | Actively maintained by Google |
